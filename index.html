Ouvrir Deriv →
2

Créer votre compte & valider l'email

Entrez votre adresse email sur la page Deriv. Vérifiez votre boîte mail, cliquez sur le lien de confirmation, puis définissez votre mot de passe sécurisé.

3

Ajouter un Compte Réel sur Deriv

Depuis votre tableau de bord Deriv, cliquez sur "Ajouter un compte réel" et sélectionnez le type "Options & Multiplicateurs". Complétez vos informations personnelles. Votre numéro de compte CR… est votre ID principal.

4
Optionnel — MT5

Si vous souhaitez utiliser MetaTrader 5, créez un compte de type "Derived" ou "Financial". Vous recevrez vos identifiants MT5 (login + mot de passe serveur).
